## Практическая работа 2: Разработка онлайн-теста с проверкой ответов

**Описание выполненной работы:**  
Создан онлайн-тест из 10 вопросов разных типов:  
- 4 вопроса с одним правильным ответом (single)  
- 3 вопроса с несколькими правильными ответами (multiple)  
- 3 вопроса с вводом текстового/числового ответа (text)  

Реализованы:  
- таймер на 10 минут;
- прогресс-бар;  
- подсчет баллов и процента правильных ответов;  
- страница результатов с разбором ошибок; 

**Файл:** `/module_1/Гиниятуллина_практика2.html`

## Практическая работа 3: Создание образовательной веб-игры

**Описание выполненной работы:**  
Создана образовательная игра-тренажёр:  
- игроку даются катеты прямоугольного треугольника (в ответах целочисленная гипотенуза);  
- нужно ввести гипотенузу и нажать «Проверить» или Enter;  
- три уровня сложности (Легко / Средне / Сложно);  
- таймер на 60 секунд;  
- система очков с бонусом за серию правильных ответов;  
- таблица рекордов.
